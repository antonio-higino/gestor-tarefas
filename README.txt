Esta é a aplicação de gestão de tarefas mostrada no PDF(aplicação exemplo), implementada em Java 11, 
sem front-end e imprimindo no navegador, com auxílio do servidor fornecido pelo Spring Boot e Spring Web.

--Passos para executar a aplicação com Spring Boot--

1. Instale o Maven, disponível em: https://maven.apache.org/;
	
2. Importe o projeto na IDE de escolha e selecione Run (para GestorTarefasApplication.java, 
presente no pacote "com.test.gestortarefas").
